--- title: Wellidate - rules ---
Wellidate - available rules 

required    - Require element to be selected/checked and it's value to not be empty.
equalto     - Require element value be the same as another element value.
length      - Require a given minimum length of characters to be entered.
minlength   - Require at least given number of characters to be entered.
maxlength   - Require element value to not exceed a given length of characters.
email       - Require a valid email address.
integer     - Require an integer value.
number      - Require a decimal number.
digits      - Require value to be digits only.
date        - Require value to match an ISO date format.
range       - Require a numeric value to be in a given range.
min         - Require a given minimum value.
max         - Require a maximum value to not be exceeded.
greater     - Require element value to be greater than a given minimum value.
lower       - Require element value to be lower a given maximum value.
step        - Require element value to be divisble by a given step value.
filesize    - Require selected file size to not exceed a given maximum number of bytes.
accept      - Require selected files to be in acceptable mime type or extension.
regex       - Require element value to match a pattern.
remote      - Require element value to pass a remote check for validity.
