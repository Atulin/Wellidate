--- title: Wellidate - api ---
Wellidate - api methods 

form     - Validates all or selected elements and shows the errors to the user.
isValid  - Validates all or selected elements without showing them to the user.
validate - Validates all or selected elements and returns the validation results for each element without showing them to the user.
rebuild  - Rebuilds wellidate instance to add or remove any dynamic elements in the container.
apply    - Applies specified validation results on the container.
reset    - Resets all validation state for the container.
