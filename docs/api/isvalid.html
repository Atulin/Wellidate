Wellidate - isValid method
Javascript 

Validates all or selected elements without showing them to the user.
isValid(...filter: string[]): boolean
    filter: list of element selectors for which validity check should be performed, no parameter implies no filter.

return: true if all validated elements are valid, otherwise false



document.querySelector("form").addEventListener("click", function (e) {
    if (e.target.classList.contains("check")) {
        if (new Wellidate(this).isValid("#name")) {
            e.target.innerText = "Check username (last check was valid)";
        } else {
            e.target.innerText = "Check username (last check was invalid)";
        }
    }
});
