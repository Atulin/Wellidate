Wellidate - validate method
Javascript 

Validates all or selected elements and returns their results.
validate(filter: string[]): object
validate(...filter: string[]): object
    filter: list of element selectors for which validity check should be performed, no parameter implies no filter.

return: validation result in format
{
    isValid: boolean,
    valid: [
        {
            validatable: object
        }
    ],
    invalid: [
        {
            method: string,
            message: string,
            validatable: object
        }
    ]
}



document.querySelector('form').addEventListener('click', function (e) {
    if (e.target.classList.contains('validate')) {
        console.log(new Wellidate(this).validate());
    }
});
