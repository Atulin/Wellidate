Wellidate - apply method
Javascript 

Applies specified validation results on the selected elements.
apply(results: object): void
    results: element selectors for which specific validation results should be applied.



document.querySelector('form').addEventListener('click', function (e) {
    if (e.target.classList.contains('apply')) {
        new Wellidate(this).apply({
            '#name': {
                error: 'Enter a unique name'
            },
            '#surname': {
                success: ''
            },
            '#birthday': {
                reset: ''
            }
        });
    }
});
