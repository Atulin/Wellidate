--- title: Wellidate - installation ---
Installation steps

Install wellidate package from npm. 

> npm i wellidate

Include wellidate scripts to your project and your page. 

<html>
    <body>
+       <script src="/scripts/wellidate.js"></script>
    </body>
</html>

Initialize Wellidate instances after rendering html 

<html>
    <body>
        ...

        <script src="/scripts/wellidate.js"></script>
        <script>
+           [].forEach.call(document.getElementsByTagName("form"), form => {
+               new Wellidate(form);
+           });
        </script>
    </body>
</html>
